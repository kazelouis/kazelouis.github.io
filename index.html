    • Deep Learning Kaggle Competition

    In 2024, I assembled a team of three engineers and participated in a Kaggle competition hosted by Harvard Medical School. We developed a deep learning model that classifies electroencephalogram (EEG) spectrogram data into six different classes of brain activities. The Convolutional Neural Network analyzes neural signals from patients and conducts multi-class classification with a 94% training accuracy.

    AI | DL | EEG | SIGNAL PROCESSING

    Dynamic Medical Lung Phantom for Radiotherapy

    I collaborated with a multidisciplinary team of six engineers to create a medical lung phantom that simulates the movement of lung tumors in real-time. After eight months of designing, programming, constructing, and testing, we developed a functional medical phantom that moves with 6-degrees of freedom. It includes an integrated control system that allows for precise manipulation of tumor movement. I contributed to developing the user interface, programming user validation functions on Matlab, and conducting integration testing to ensure synchronization between software and hardware. I also coordinated the team workflow by creating an agile project management master plan.

    BIOMED | SW | HW | MATLAB | UI | ARDUINO

    Autonomous Vehicle Path Navigation System on FPGA

    Using Verilog and embedded system design principles, I developed an autonomous vehicle path navigation system on a Field-Programmable Gate Array (FPGA). The system allows an object to navigate through an obstacle-filled map and displays the fastest path on a VGA display screen. The path is optimized through utilizing behavioral modeling algorithms. To achieve this, I iterated through several different Finite State Machine designs and multiplexers. Compilation trial times were minimized through utilizing ModelSim for testing and analysis.

    AI | HW | FPGA | VERILOG | FSM
